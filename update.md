# 网站日常更新规则
（1）当用户输入“更新网站”或者进行日常网站跟新时，
  AI Agent 运行本地的黄金交易算法，获得当天最新的结果，然后更新网站。
（2）网站更新方式参考README.md中的建议
（3）进行日常更新的时候不要修改交易算法的策略，直接使用已有的交易算法，算法在 ./research 目录下
（4）如果在日常更新期间AI Agent发现交易算法存在问题或者随着时间推移算法已经失效，将情况反馈给用户，不要直接修改算法
（5）跟新网站时保持网站风格统一。让网站在电脑和手机浏览器上都体现用户友好
（6）只有当用户要求修改或者优化交易算法的时候才修改算法，只有当用户要求修改网站样式的时候才修改网站样式。
